## Лабораторная работа 1
Отказ от using namespace std: Во всех файлах удален в соответствии с выбранным кодстайлом.  
Инкапсуляция: Каждая задача теперь состоит из трех файлов.  
Библиотеки: Добавлены недостающие библиотеки для работы с бесконечностью и для вычислительных функций.  
Форматирование: Все открывающие фигурные скобки { перенесены на новые строки.  
Коды возврата: В функции main добавлен явный return 0; в конце, а в некоторых проверках на ошибки  return 0.

Функция HasEnoughNumbersInLine, которая проверяет наличие ровно N чисел и отсутствие букв.

Появилась проверка на наличие в файле неиспользованных символов после считывания всех данных.
